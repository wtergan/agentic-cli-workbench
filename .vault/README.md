# Dotfiles Vault

Operational memory for the agentic-cli-workbench repo. Keep implementation details, planning
state, evidence, and handoff packets here instead of scattering them through
chat history.

## Structure

- `PLAN.md` - roadmap, active plan index, goal runs, risks, and knowledge links.
- `plans/` - executable feature plans.
- `research/` - local code research and external research.
- `decisions/` - durable ADR-style decisions.
- `solutions/` - reusable fixes and implementation patterns.
- `encounters/` - recurring blockers, root causes, and prevention notes.
- `goals/` - thin orchestration overlays for native `/goal` or multi-plan work.
- `visuals/` - optional self-contained diagrams or HTML companions.

## Active Focus

- `agentic-cli-workbench`: curate a public showcase repo from the private
  dotfiles workflow without leaking private host state, account data, or live
  agent configuration.
